# CoffeeBase

## Project info
Project was made for training and portfolio purposes.

CoffeeBase is simple android app, it uses coffeebase API webservice (available in another repository)
for functionalities:
* Create new coffee with name, roaster, rating, image
* Look over added coffees
* Delete coffee
* Add coffee to favoutrites

It presents saved coffees in elegant form, using some of Android features like Card View and Recycler View.

## Technology
* Java
* Android (Retrofit2)
